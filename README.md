# Sales and Call Center Dashboard

## Project Overview
Interactive Excel dashboard designed to analyze sales performance, call center operations, agent productivity, and state-wise metrics.

## Features
- KPI Cards
- Funnel Analysis
- State-wise Map Visualization
- Agent Performance Tracking
- Trend Analysis
- Interactive Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Map Charts
